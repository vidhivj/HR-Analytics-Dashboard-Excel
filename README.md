# HR Analytics Dashboard (Excel)

## Overview
This project is an interactive HR Analytics Dashboard created using Microsoft Excel. It helps analyze employee attrition, workforce distribution, salary patterns, age groups, job satisfaction, and business travel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Dataset
- Total Employees: 1470
- Departments: 3

## Dashboard Features
- Employee Distribution
- Attrition Analysis
- Gender Distribution
- Age Group Analysis
- Salary Distribution
- Job Satisfaction
- Business Travel Analysis
- Interactive Filters
